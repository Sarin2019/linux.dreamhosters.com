Table of Contents

Basic listing

List all files and directories in a vertical format

[server]$ ls -la

Show file types and directories

Directories are noted by trailing slash and symbolic links with @.

[server]$ ls -laF

List only directories

[server]$ du -hx --max-depth 1

List files and directories by timestamp

Lists all files and directories in the current directory in order.

[server]$ ls -lat

Listing by size

List files and sort by size

[server]$ ls -lahS

This example only outputs the size and files.

List all files, directories and sub-directories in bytes

Change the letter 'b' to 'k' if you do not want to show results in kilobytes.

[server]$ du -ab | sort -nr

List all directories and sub-directories in kilobytes

[server]$ du -k | sort -nr | more | head

Listing directories

List all directories over a certain size

Use grep K for kilobytes. Use grep M for megabytes. Use grep G for Gigabytes. 

[server]$ du -hx --max-depth=1 |grep M
