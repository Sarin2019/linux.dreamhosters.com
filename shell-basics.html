Root or user prompt

When you log in you'll run commands as a user or root. You can tell by the symbol after the username. For example:

Root prompt

The root user will always have a # sign after its name (which is also root). This example also assumes you're logged into a server, not your local computer:

root@server:~#
Note that the ~ symbol means you're in that user's home directory.

User prompt

If you're logged in as a user the prompt ends with a $ sign.

user@server:~$
Note that the ~ symbol means you're in that user's home directory.
